# Sistema de Comandas

Projeto idealizado para a disciplina de Engenharia de Software 3
da FATEC - Mogi das Cruzes.

## Dependências

1. Git
2. Maven

## Rodar a aplicação

Após clonar o repositório entre na pasta do projeto

```
$ cd SistemaComandas
```

Rode o comando:

```
$ mvn jetty:run
```

Acesse o endereço `localhost:8080` no navegador
